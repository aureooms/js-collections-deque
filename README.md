[js-collections-deque](http://aureooms.github.io/js-collections-deque)
==

Python's deque data structure for JavaScript. Parent is
[js-data-structures](https://github.com/aureooms/js-data-structures).

```js
deque( 'abcde' , 3 ) ; // 'cde'
```
